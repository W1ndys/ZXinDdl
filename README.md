# ZXinTools

- 知新平台作业截止提醒（未开发）
- 知新平台作业查看成绩（已开发）
- 知新平台作业客观题百分百正确自动化作答（已有思路，已实现半自动，未开发）

## 使用方法

### 查看成绩

1. 修改 `config.json` 中的账号密码
2. 运行 `get_score.py`
3. 结果会保存到 `socre_info/course_summary.txt` 文件中
